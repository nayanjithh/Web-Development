HTML Structure and Special Characters

An HTML document is made up of several key parts:

  • <!DOCTYPE html> - Declares the document type and version of HTML.
  • <html> - The root element that wraps all the content of the document.
  • <head> - Contains meta-information about the document, such as its title and links to stylesheets.
  • <body> - Contains the visible content of the document, such as text, images, and links.
  • </> - Closing tags that signify the end of an element.

Using Special Characters in HTML

In HTML, certain characters are reserved for the markup itself. If you want to display these characters as text, you must use special codes called HTML entities.

  • & - The ampersand character is used to start an HTML entity. To display it as text, use &amp;.
  • < - The less-than symbol is used in tags. To display it as text, use &lt;.
  • > - The greater-than symbol is also used in tags. To display it as text, use &gt;.
  • ™ - The trademark symbol can be displayed using &trade;.
  • © - The copyright symbol can be displayed using &copy;.
  • ® - The registered trademark symbol can be displayed using &reg;.
